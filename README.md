# AI CCTV Surveillance System

## 📌 Overview

A real-time AI-based surveillance system designed to automate CCTV monitoring and detect suspicious activities.

---

## 🧠 Tech Stack

* Python
* OpenCV
* YOLOv8
* MTCNN
* FaceNet
* Streamlit

---

## ⚙️ Features

* Person detection using YOLOv8
* Face detection using MTCNN
* Face recognition using FaceNet
* Abnormal activity detection
* Real-time alert system

---

## 🔄 System Workflow

Camera → YOLO Detection → Face Detection → Face Recognition → Activity Analysis → Alert
